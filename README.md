# FinFlow Portal

A premium SaaS Billing Portal & Invoice Generator.

## Run Locally
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Features
- Pricing switcher (Monthly/Annual) with animated cards & checkout modal
- Interactive invoice builder with line items, tax, discount codes & print/PDF
- Transaction history with search & filter
- Revenue/Expenses analytics chart (Recharts)
- Full dark mode toggle
